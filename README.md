# Family Meal Planner

A multilingual Android meal planning application that helps families organize their recipes, plan meals, and generate shopping lists.

## Features

- **Recipe Collection**: Browse and manage recipes across multiple cuisines (Asian, Mediterranean, Mexican, Italian, French, and more)
- **Meal Planning**: Plan your weekly meals with an intuitive calendar interface
- **Shopping Lists**: Automatically generate shopping lists from your meal plans
- **Multilingual Support**: Available in English, Russian, and Romanian
- **Customizable**: Adjust font styles, accent colors, and accessibility settings
- **Offline Access**: All recipes stored locally for offline use

## Tech Stack

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Architecture**: MVVM with Clean Architecture
- **Database**: Room
- **Dependency Injection**: Hilt
- **Build System**: Gradle with Kotlin DSL

## Getting Started

1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle dependencies
4. Run the app on an emulator or physical device

## Requirements

- Android Studio Hedgehog or later
- Minimum SDK: 26 (Android 8.0)
- Target SDK: 34 (Android 14)
